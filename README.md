# 电商用户行为分析与智能营销Agent

这是一个基于LangChain和DeepSeek大模型的电商用户全生命周期智能营销Agent系统。它解决了传统电商运营中用户分群粗糙、营销文案千人一面、活动效果无法实时优化的核心痛点。

## 系统架构

系统采用多Agent协作架构，包含四个核心模块：

1. **数据采集Agent**：自动加载和预处理用户浏览、加购、下单等行为数据
2. **用户画像Agent**：通过长链推理生成包含消费能力、兴趣偏好、购买周期的360度用户标签
3. **策略生成Agent**：根据不同用户分群自动设计个性化营销方案
4. **效果反馈Agent**：实时监控转化数据并迭代优化策略
